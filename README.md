# 🚑 EMS Минск — PWA для скорой медицинской помощи

Прогрессивное веб-приложение для врачей и фельдшеров скорой помощи.

## 📁 Структура проекта

```

ems-minsk/
├── index.html              # Главная страница
├── manifest.json           # PWA-манифест
├── sw.js                   # Service Worker
├── css/
│   ├── main.css           # Точка входа (импорт всех стилей)
│   ├── variables.css      # CSS переменные (цвета, отступы)
│   ├── base.css           # Базовые стили и сброс
│   ├── header.css         # Верхняя панель
│   ├── sidebar.css        # Боковое меню
│   ├── home.css           # Главная страница
│   ├── documents.css      # Приказы, рекомендации, справочники
│   ├── calculators.css    # Калькуляторы
│   ├── modal.css          # Модальное окно
│   └── responsive.css     # Адаптивность
├── js/
│   ├── app.js            # Главный модуль
│   ├── data.js           # Данные
│   ├── utils.js          # Утилиты
│   └── db.js             # IndexedDB
└── README.md

```

## 🎨 Модульная архитектура стилей

Стили разделены на логические модули для:
- **Удобства поддержки** — каждый файл отвечает за свою часть
- **Масштабируемости** — легко добавлять новые разделы
- **Переиспользования** — общие переменные и компоненты
- **Автоматического подключения** — main.css импортирует все стили

## 📱 Функционал

- **🏠 Главная** — приветствие по времени суток, быстрая навигация, избранное
- **📋 Приказы** — документы с поиском и фильтрацией
- **📖 Клинические рекомендации** — протоколы и алгоритмы
- **🧮 Калькуляторы** — шкала Глазго, CURB-65
- **📚 Шпаргалки** — лекарства, нормы, критерии

## ⚙️ Технологии

- Vanilla JS — без фреймворков
- IndexedDB — хранение избранного
- Service Worker — офлайн-режим
- CSS Variables — темизация
- Hash Router — SPA-навигация

## 🚀 Установка

1. Скопируйте все файлы на веб-сервер
2. Откройте `index.html` в браузере
3. Для PWA — установите приложение на главный экран

## 🔧 Разработка

При добавлении новых стилей:
1. Создайте файл в папке `css/`
2. Импортируйте его в `main.css`
3. Используйте переменные из `variables.css`
