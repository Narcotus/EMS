# EMS Минск — PWA для бригад скорой медицинской помощи

Офлайн PWA-приложение в стиле Material Design 3 (бордовый акцент #800020).

## Разделы
- 📋 Приказы (фильтр по году, сортировка, поиск)
- 📖 Клинические рекомендации (категории, сортировка, поиск)
- 🧮 Калькуляторы (GCS, Wells, дозировки, HAS-BLED, CHA₂DS₂-VASc, SOFA)
- 📚 Справочная информация

## Запуск
```bash
cd ems-minsk
python3 -m http.server 8000
```

Откройте [http://localhost:8000](http://localhost:8000)
